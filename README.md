# ProDt task

## Run   

Clone the project  

~~~bash  
  git clone https://github.com/shindekalpesharun/flutter-prodt-task
~~~

Go to the project directory  

~~~bash  
  cd flutter-prodt-task
~~~

Install dependencies  

~~~bash  
flutter pub get
~~~

run the app

~~~bash  
flutter run -d device_id
~~~
